# 🚀 Летающий в пустоте

Мультиплеерная 3D-игра на Three.js с чатом, монетами и ускорением.

## Управление
- **WASD / Стрелки** — движение
- **Shift** — ускорение
- **C** — смена камеры
- **ПКМ** — вращение камеры

## Запуск
1. Установи Node.js
2. Установи зависимости: `npm install`
3. Запусти сервер: `node server.js`
4. Открой в браузере: `http://localhost:3000`

## Технологии
- Three.js (3D-графика)
- Socket.io (мультиплеер)
- Node.js (сервер)
