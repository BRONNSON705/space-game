# 🚀 Летающий в пустоте (Space Game)

**Мультиплеерная 3D-игра на Three.js**, в которой вы управляете летающей машиной, собираете монеты в открытом космосе, соревнуетесь с друзьями по локальной сети и общаетесь во встроенном чате.

---

## 🎮 Особенности

| Фича | Описание |
|------|----------|
| **3D-мир** | Полноценное трёхмерное пространство с космическим фоном, туманностями и звёздами. |
| **Управление машиной** | WASD / стрелки — движение, Shift — ускорение, C — смена камеры. |
| **Камеры** | 3 режима: вид сзади, сверху, от первого лица. |
| **Монеты** | Собирайте монеты, они появляются с анимацией и звуком. |
| **Метеориты** | Препятствия, которые отбрасывают машину при столкновении. |
| **Мультиплеер** | Играйте с друзьями в локальной сети: все видят друг друга. |
| **Чат** | Встроенный чат с системными уведомлениями (подключение / отключение игроков). |
| **Таблица лидеров** | Отображает количество собранных монет у всех игроков в реальном времени. |
| **Список игроков** | Отдельное окно со списком всех активных игроков. |

## 📦 Установка и запуск

### 1️⃣ Требования

- Установленный [Node.js](https://nodejs.org/) (версия 16 или выше)
- Любой современный браузер (Chrome, Firefox, Edge)

### 2️⃣ Скачать проект

```bash
Скачать ZIP-архив и извлечь данные на рабочий стол.
```

### 3️⃣ Установить зависимости

```bash
npm install
```

### 4️⃣ Запустить сервер

```bash
Открыть cmd через Win+R.

Ввести данную команду: cd C:\Users\User\Desktop\space-game

После данную команду: node server.js
```

### 5️⃣ Открыть игру

Открой браузер и перейди по адресу:

```
http://localhost:3000
```

## 🕹 Управление

| Клавиша / Действие | Эффект |
|-------------------|--------|
| `W` / `↑` | Движение вперёд |
| `S` / `↓` | Движение назад |
| `A` / `←` | Поворот налево |
| `D` / `→` | Поворот направо |
| `Shift` | Ускорение (с перезарядкой) |
| `C` | Смена камеры (3 режима) |
| `ПКМ` + мышь | Вращение камеры (в режимах сзади / сверху) |
| `Колёсико мыши` | Приближение / отдаление |
| `Enter` (в чате) | Отправить сообщение |

## 🔗 Как играть с друзьями

1. Убедитесь, что вы в одной локальной сети (один Wi-Fi / роутер).
2. **Первый игрок** запускает сервер и узнаёт свой IP-адрес:
   - В командной строке введите `ipconfig`
   - Найдите `IPv4-адрес` (например, `192.168.1.100`)
3. **Остальные игроки** открывают браузер и вводят:

```
http://192.168.1.100:3000
```

## 🧩 Возможные проблемы и решения

| Проблема | Решение |
|----------|---------|
| Ошибка `node: not found` | Установи Node.js с официального сайта |
| Ошибка `cannot find module` | Запусти `npm install` в папке проекта |
| Второй игрок не подключается | Проверь IP-адрес, отключи брандмауэр на порту 3000 |
| Нет звука при сборе монет | Убедись, что в браузере разрешён звук |
