# 🚀 Летающий в пустоте (Space Game)

**Мультиплеерная 3D-игра на Three.js**, в которой вы управляете летающей машиной, собираете монеты в открытом космосе, соревнуетесь с друзьями по локальной сети и общаетесь во встроенном чате.

---

## 🎮 Особенности

| Фича | Описание |
|------|----------|
| **3D-мир** | Полноценное трёхмерное пространство с космическим фоном, туманностями и звёздами. |
| **Управление машиной** | WASD / стрелки — движение, Shift — ускорение, C — смена камеры. |
| **Камеры** | 3 режима: вид сзади, сверху, от первого лица. |
| **Монеты** | Собирайте монеты, они появляются с анимацией и звуком. |
| **Метеориты** | Препятствия, которые отбрасывают машину при столкновении. |
| **Мультиплеер** | Играйте с друзьями в локальной сети: все видят друг друга. |
| **Чат** | Встроенный чат с системными уведомлениями (подключение / отключение игроков). |
| **Таблица лидеров** | Отображает количество собранных монет у всех игроков в реальном времени. |
| **Список игроков** | Отдельное окно со списком всех активных игроков. |

## 🛠 Технологии

| Технология | Назначение |
